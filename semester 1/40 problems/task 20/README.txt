Каждое число представляет собой массив байт.
Первый байт массива содержит код знака числа: 0 - "+", 225 - "-",
а остальные байты- значащие двоичные цифры числа.
В программе разработать процедуры алгебраических операций с двумя целыми числами,
каждое из которых представлено массивом байтов.
Операции:
1. Сложение 
2. Вычитание 
3. Умножение 
4. Деление на цело 
При выполнении операций учитывается возможность переполнения.
