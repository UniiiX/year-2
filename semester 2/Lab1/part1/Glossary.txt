User - a person who interacts with the system, hence has the ability/access to do so.

Registered User -  already who has already signed up.

Guest User - not registered yet user.

Media - photo/video.

News Feed - list of updates on people and groups you follow.

Message - consists of text, media, songs, attached messages.

Conversation - chat in which users can write messages.

Interlocutors - set of users who participate in a conversation.

Global Playlist - all songs that are added by the user are automatically placed in it.

Music - set of playlists + global playlist.

General Group - generalized term which is used to describe group of either type Business, Thematic or Public Page.

Last time seen - time of the last user's log out.

Follow = Subsrcribe to.

Sign in = log in.

Sign up = register.
